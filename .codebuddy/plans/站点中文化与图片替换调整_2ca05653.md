## 用户需求

1. **站点中文化**：将博客站点从英文切换为中文界面

- HTML lang 属性改为 zh-CN
- 导航菜单中文化
- 首页文案中文化
- 关于页面文案中文化
- 文章详情页文案中文化
- Footer 区域中文化

2. **图片替换**：使用用户提供的图片替换占位图

- 3 张用户图片已存在于 `/public/images` 目录
- 需要更新 3 篇文章的 cover 字段

### 实现细节

1. **语言属性修改**

- `<html lang="en">` → `<html lang="zh-CN">`
- Metadata 中 title/description 改为中文

2. **导航中文化**

- Home → 首页
- Categories → 分类
- About → 关于

3. **首页文案**

- Welcome to my blog → 欢迎来到我的博客
- Sakura Blog → 樱花博客
- Latest Posts → 最新文章
- Browse by Category → 按分类浏览
- Marquee 关键词: Technology→技术, Anime→动漫, Life→生活, Learning→学习, Coding→编程, Design→设计

4. **关于页面**

- Skills and Interests → 技能与兴趣
- Get in Touch → 联系我
- Send Message → 发送消息

5. **文章详情页**

- Back to posts → 返回文章列表
- Related Posts → 相关文章

### 构建验证

修改完成后执行 `pnpm build` 验证无错误。
