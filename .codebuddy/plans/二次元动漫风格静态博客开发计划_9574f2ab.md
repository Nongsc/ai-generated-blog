## 产品概述

一个二次元、日常动漫风格的静态博客系统，采用清新柔和的色调设计，支持 Markdown 文章发布与展示、分类标签管理、个人介绍页面。

## 核心功能

1. **文章展示系统**

- Markdown 格式文章渲染
- 文章列表展示（首页）
- 文章详情页阅读体验

2. **分类与标签系统**

- 文章按分类筛选
- 文章按标签筛选
- 分类/标签列表展示

3. **关于我页面**

- 个人介绍展示
- 内容通过配置文件管理

## 开发约束

- 不使用 Emoji 表情
- 每完成功能需单元测试
- 每完成功能需记录开发历史
- 遵循代码规范、文档规范、Git 规范
- 保持简洁文件结构
- 调试两次失败需重写功能

## 技术栈选择

### 前端框架

- **React 18 + TypeScript**: 采用函数组件和 Hooks，符合现代 React 最佳实践
- **Next.js 15**: App Router 架构，支持静态生成和服务端渲染
- **Vite**: 快速构建工具（如使用纯 React）

### 样式方案

- **Tailwind CSS**: 原子化 CSS，便于快速实现二次元风格设计
- **Magic UI**: 可复制粘贴的精美组件库，提供丰富的动画效果和视觉效果
- **自定义配色**: 清新柔和色调（粉色、淡蓝、薄荷绿为主色调）

### UI 组件库

- **Magic UI**: 基于 shadcn/ui 理念的组件库，提供：
- 动画组件：Marquee、Animated List、Text Animate
- 特效组件：Border Beam、Magic Card、Meteors
- 布局组件：Bento Grid、Dock
- 按钮组件：Shimmer Button、Rainbow Button
- 安装方式：`pnpm dlx shadcn@latest add @magicui/{component}`

### Markdown 处理

- **react-markdown**: React Markdown 渲染组件
- **remark-gfm**: GitHub Flavored Markdown 支持
- **rehype-highlight**: 代码高亮
- **gray-matter**: frontmatter 解析

## 实现方案

### 架构设计

```
静态文件驱动 -> Markdown 文章 + JSON 配置
    ↓
Next.js 构建时处理 -> 静态生成页面
    ↓
React 应用运行时 -> 动态加载渲染
```

### 性能考量

- 构建时预解析 Markdown，减少运行时计算
- 按需加载文章内容，优化首屏加载
- 图片懒加载，提升页面性能
- 路由懒加载，减少初始包体积

## 设计风格

采用 **清新动漫风格 (Soft Anime Style)**，以柔和的色调和圆润的设计元素营造温馨舒适的阅读体验。

### 页面规划

#### 1. 首页 (Home)

- **顶部导航栏**: 固定定位，半透明背景，Logo + 导航链接
- **横幅区域**: 柔和渐变背景，博客标题，简短介绍
- **文章列表**: 卡片式布局，展示封面图、标题、摘要、分类标签
- **底部区域**: 版权信息，社交链接

#### 2. 文章详情页 (Post Detail)

- **返回按钮**: 返回首页
- **文章头部**: 标题、日期、分类、标签
- **文章内容**: Markdown 渲染区，优化阅读体验
- **相关文章**: 同分类/标签推荐

#### 3. 分类/标签页 (Category/Tag)

- **筛选区域**: 分类/标签选择器
- **文章列表**: 筛选结果展示

#### 4. 关于我页面 (About)

- **头像区域**: 圆形头像，柔和阴影
- **个人信息**: 姓名、简介、社交链接
- **技能/兴趣**: 标签云展示
